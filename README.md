# [www](www.huggins.io)

The code behind my website.

## Development
1. Download the repository
```text
git clone https://github.com/kjhx/www.git
```
2. Install dependencies
```text
yarn
```
3. Start the dev server
```text
yarn dev
```

## Tools
* Built with NextJS, by Vercel
* Set in Inter, by Rasmus Andersson
* Styled with TailwindCSS, by Tailwind Labs
* Hosted on Vercel, previously ZEIT Now
